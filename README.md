Introduction
Bootstrap Basics is a simple website built to demonstrate foundational understanding of Bootstrap components and layout structure. The project was created in VS Code using a combination of class notes, documentation from getbootstrap.com, and assistance from Copilot.

The page includes a mix of Bootstrap utilities, responsive layout elements, and styled components. At the bottom of the page, you’ll find a personal touch: a photo of Mary’s pup wearing a raincoat at just a few months old back in 2022.

This project served as hands‑on practice in navigating Bootstrap’s grid system, spacing utilities, and component classes while building a clean, responsive layout.

Features
Responsive layout built with the Bootstrap grid system

Use of Bootstrap components such as navbars, cards, buttons, and containers

Custom styling layered on top of Bootstrap defaults

Personal image included at the bottom of the page

Clean, beginner‑friendly structure suitable for future expansion

Prerequisites
Before installing, ensure you have the following installed on your machine:

Git

Node.js and npm

A code editor such as VS Code

Installation
To access and run the project locally, open your Terminal and follow these steps:

bash
git clone https://github.com/mfobbsie/BootstrapBasics
cd BootstrapBasics
npm install
npm start
Usage
Once the development server starts, the site will open in your browser. You can explore the Bootstrap components, adjust the layout, or customize the styling to reflect your personal aesthetic. This project can also serve as a starting point for experimenting with more advanced Bootstrap features or integrating JavaScript functionality.
